«É inaceitável dizer que um jogador é bom negócio»

Em conversa com o Le Figaro, Laurent Koscielny deixou críticas a quem trata os jogadores «como mercadoria». Na ótica do central do Bordéus, esta paragem devia servir para as pessoas refletirem sore o mercado de transferências no futebol.

 

«É preciso voltar a confiar nos jogadores para reconstruir o futebol, para isso devem-nos tratar de forma diferente, não como mercadoria. É inaceitável dizer que um jogador é um bom negócio, ou que tem uma boa relação qualidade-preço. É uma imagem triste que se passa para os jovens», desabafou.

 

Formado no Guingamp, Koscielny assinou esta temporada pelo Bordéus, depois de nove temporadas no Arsenal. No total, soma 28 partidas pelos girondinos.
